PUVI - The Bilingual Systems Programming Language

--COMING SOON--2026--

Puvi is a modern, statically typed, compiled systems programming language written in Rust. Native Tamil & English syntax. Zero runtime overhead. Built for AI, Robotics, and High Performance.

Get Started Download Puvi v1.0 

                

            

        

    


    
    

        

            

                
Why Puvi?

                
Designed from the ground up to provide maximum performance without compromising on developer experience.

            

            
            

                

                    
Native Bilingual Syntax

                    
Write code in English or Tamil. Both syntaxes compile directly into the exact same Abstract Syntax Tree (AST). No translation layer overhead.

                

                

                    
Blazing Fast

                    
Compiled natively via LLVM. Puvi matches or exceeds the performance of C and Rust, ensuring zero runtime cost abstractions.

                

                

                    
Modern Safety

                    
Memory safety guarantees at compile time without a garbage collector. Eliminate data races and null pointer exceptions.

                

                

                    
Robotics Ready

                    
First-class support for deterministic execution and low-latency hardware control. Ideal for RTOS and embedded development.

                

                

                    
AI Friendly

                    
Built-in primitives for SIMD, matrix operations, and GPU interfacing to accelerate modern machine learning workloads.

                

                

                    
Open Source

                    
Developed entirely in Rust, Puvi is open source and driven by the community. Join us in shaping the future of programming.
